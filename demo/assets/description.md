## TurboQuant Inference Engine — Demo

This Gradio application demonstrates the TurboQuant KV Cache Compression and Semantic Search Engine.

### Three Tabs

1. **KV Cache Compression** — Simulate memory savings when compressing LLM KV caches
2. **Semantic Search** — Approximate nearest-neighbor search using compressed vectors
3. **Algorithm Visualizer** — Step-by-step walkthrough of the TurboQuant encoding pipeline
